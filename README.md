# Kdotle
Wordle for Kendrick Lamar Songs (Based on Yeezle.xyz)

A fun Wordle-like game that can be ran in any python interpreters.

There will be a random chosen Song from any of Kendrick Lamars Albums and you will have 8 guesses to guess which song it is, hints are given after every response. 

This was a little passion project made in a day or 2 made for me and my friends (we are big Kendrick Lamar fans).
